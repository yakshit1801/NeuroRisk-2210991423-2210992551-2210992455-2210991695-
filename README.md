# Stroke & Brain Clot Risk Assessment Tool (Neuro Risk)

## 📌 Project Overview
This repository contains the research paper titled **"Stroke Risk Monitoring System (AI Assisted)"**, which focuses on predicting stroke risk using healthcare parameters and AI-based interpretation.

The study presents a web-based prototype that evaluates risk levels based on clinical and lifestyle factors and provides decision-support insights.
